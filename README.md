Pizza Sales Analysis Dashboard

Project Overview
This project focuses on analyzing pizza sales data using SQL and Power BI to uncover valuable business insights such as revenue trends, customer ordering patterns, top-selling pizzas, peak order timings, and sales performance by category and size.
An interactive dashboard was created in Power BI to visualize key metrics and support business decision-making.

Dashboard Insights
Large pizzas generated the highest revenue.
Peak order timings were observed during evening hours.
Classic category pizzas contributed significantly to total sales.
A small number of pizzas contributed to most overall revenue.

orders → order_details
pizzas → order_details
pizza_types → pizzas
